# Story 4.3: Desktop App Shell

**Status**: Done
**Epic**: Epic 4 — Desktop App
**Estimate**: 3h
**Priority**: High

## User Story
As a user, I want a desktop app with a sidebar layout, header search, and routing, so that I can navigate between screens without page reloads.

## Acceptance Criteria
- [x] App layout: sidebar (left) + main content (right)
- [x] Header with title, SearchBar, settings icon
- [x] React Router setup with routes for all screens
- [x] ThemeProvider wrapping the whole app
- [x] ErrorBoundary catches render errors gracefully

## Technical Notes
- **Dependencies**: Story 4.1, Story 3.6
