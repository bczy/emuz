# Story 3.7: GameDetail Component

**Status**: Done (tests pending)
**Epic**: Epic 3 — UI Components
**Estimate**: 3h
**Priority**: Medium

## User Story
As a user, I want a detailed view of each game with full artwork, metadata, and play statistics, so that I can learn about it before playing.

## Acceptance Criteria
- [x] Large cover image (hero)
- [x] Metadata section: title, platform, developer, publisher, year, genre, description
- [x] Play statistics: total time, last played, play count
- [x] Launch button (prominent, emerald)
- [x] Favorites toggle
- [x] Edit metadata button
- [ ] Tests: renders with and without metadata

## Technical Notes
- **Dependencies**: Story 3.2
- **PRD ref**: US-3.3
