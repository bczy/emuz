# Story 5.4: Mobile Screens

**Status**: Done
**Epic**: Epic 5 — Mobile App
**Estimate**: 8h
**Priority**: High

## User Story
As a mobile user, I want all main screens available on mobile, so that I can manage and launch my games from my phone or tablet.

## Acceptance Criteria
- [x] `HomeScreen` — widget grid (Daijishou style), scrollable
- [x] `LibraryScreen` — GameGrid with filter toolbar
- [x] `PlatformsScreen` — PlatformCard grid with wallpapers
- [x] `GenresScreen` — GenreList + game grid
- [x] `CollectionsScreen` — collections list + game grid
- [x] `GameDetailScreen` — full-screen with hero art, launch button
- [x] `SettingsScreen` — emulator config, theme, storage
- [x] `EmulatorConfigScreen` — per-platform emulator setup

## Technical Notes
- **Dependencies**: Story 5.3, Story 3.3 through 3.11
- **PRD ref**: Epic 3 (all US-3.x), US-4.1
